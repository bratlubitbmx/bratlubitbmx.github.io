Всё о маунтинбайке

Данный сайт посвящён одному из видов экстремальных видов спорта - маунтинбайку.

В настоящее время существует много видов маунтинбайков.

  • Байк для даунхила,фрирайда и норд - шора.

  • Байк для кросс - кантри.

  • Байк для стрита.

  • Байк для дёрта.

  • Байк для езды за пивом.
